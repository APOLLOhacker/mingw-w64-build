# mingw-w64-build
MinGW-w64 cross compiler build script for i686 (Win32) and x86_64 (Win64).

## Package
* Binutils 2.28
* GCC 7.1.0
* GMP 6.1.2
* MPFR 3.1.5
* MPC 1.0.3
* isl 0.18
